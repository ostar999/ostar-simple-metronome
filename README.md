# 🎵 ostar-simple-metronome

一个功能完整的网页节拍器，纯 HTML/CSS/JS，浏览器打开即用。

## ✨ 功能

- **精准节拍** — Web Audio API look-ahead 调度，时序不漂移
- **自定义强弱拍** — 每拍可设 强 / 次强 / 弱 / 休止，点击圆点切换
- **突出次强开关** — 关闭时次强跟随弱拍响度
- **独立响度调节** — 强/次强/弱三档各自手动输入（0.0–1.0）
- **音符时值** — 4分 / 8分 / 16分 / 32分音符，按基本乐理计算（3/8+8分=3声，4/4+8分=8声）
- **多种拍号** — 2/4、3/4、4/4、3/8、6/8、12/8 快速切换
- **速度预设** — Largo~Presto 七档，中英文对照
- **四种音色** — 经典 / 木鱼 / 电子 / 响棒，Web Audio 合成
- **Tap Tempo** — 打点测速
- **摆锤动画** — 大屏显示，随节拍摆动
- **夜间模式** — 🌙/☀️ 一键切换
- **键盘快捷键** — Space 启停、方向键调 BPM、T 打点
- **自动记忆** — 所有设置 localStorage 持久化

## 🚀 使用

```bash
open metronome.html
```

## ⌨️ 快捷键

| 键 | 功能 |
|----|------|
| Space | 播放 / 暂停 |
| ↑↓ | BPM ±1 |
| ←→ | BPM ±1（Shift +10） |
| T | Tap Tempo |

## 🛠 技术

- 零依赖，单文件 2000 行
- Web Audio API 合成 + 调度
- localStorage 持久化
- CSS 变量 + 多级响应式
